DeFi协议越来越多地采用「代币空投」的方式，以分散治理并促进早期用户的参与，项目方通常会进行“追溯性空投”，这意味着他们会根据你过去与其协议或系统交互的多少来分配奖励。

在这篇文章中，我们将概述一些可能会进行空投的协议。

目标：确定未发行代币的协议，以捕获未来可能的空投机会。

技能：初级

努力：中等

经验总结

根据我们在过去一年中看到的情况，我们可以合理地得出一个结论：「空投」正在成为年轻 DeFi 项目的标准操作实践。从项目方的角度来看，优化这些空投还有一些工作要做（比如让它们抵抗女巫攻击）。

问题的关键是？大多数人没有大量的时间和无限的资源来将活动分散到每个最终可能会推出代币的 DeFi 项目中。

因此，重点就在于效率：确定要关注哪些项目（例如，给定协议是否有非常小的机会在6-12个月内推出代币？）以及规模（即存款多少以平衡风险和潜在的回报）。

就第一点而言，通过一点研究，你通常可以找到团队/社区的代币计划来指导你的工作。例如，ParaSwap 最近宣布它没有空投计划。相反，MetaMask团队最近透露其对最终发布原生代币的想法持开放态度。

因此，作为一个例子，你可能想开始通过MetaMask 执行一些加密资产交易。当然，我们并不能保证代币空投就一定会到来，但至少现在我们知道了，它存在着可能。

接下来，为了获得空投资格，你需要和这些未发行代币的协议进行多少次（或量）的交互？

这将因项目而异，但DeFi早期的黄金法则是“决不要存入超过你承受能力的加密资产。”

好消息是什么？正如The Daily Ape创始人Darren Lau在上个月指出的那样，即使只是一点点的钱/活动，也足以让你获得空投的资格。

25个可能会有空投的加密项目

下面是我们V2版空投指南收录的加密项目列表，这些项目有可能在不久的将来推出自己的代币并进行追溯性空投。

免责声明：我并不确定它们是否会进行空投，这完全是基于我（作为一名活跃的加密用户和研究人员）的经验得出的观点。

1、Element Finance

描述：Element Finance是一个“针对固定收益率和可变收益率市场的开源协议”。

资源：网站|文档‌|Twitter‌|Discord‌

理由：Element Finance已经创建了一个治理模型，这表明该项目必然会发布一个原生代币。

空投策略：Element Finance的旗舰产品是其Earn应用程序和Save应用程序，使用这两种产品中的任何一种，都可能让你有资格获得未来的空投。

2、Hop Protocol

描述：Hop Protocol是一种代币桥协议，其用于在区块链和rollup二层网络之间轻松发送ETH和ERC-20代币，它目前支持跨以太坊、Arbitrium、Optimism、Polygon以及xDai这几条链。

资源：网站‌|文档‌|Twitter‌ | Discord‌

理由：关于潜在的Hop代币，该项目团队之前已说过“任何关于该协议的官方公告，Hop团队都将直接发布在官方的Hop频道。”

如果Hop没有发布代币的计划，他们可能会像ParaSwap一样发布官方公告。

空投策略：访问hop.exchange，然后（1）使用发送仪表板将资产从一条链跨到另一条链，或者（2）成为Hop池子的流动性提供者（LP）。

3、DeFi Saver

描述：DeFi Saver是“一站式仪表板，用于创建、管理和跟踪你的DeFi头寸。”

资源：网站‌|文档‌|Twitter‌ | Discord‌

理由：到目前为止，DeFi Saver一直在使用一个原生代币，但这是一个策展代币，持有者可以用其投票和优先考虑新的协议整合。

空投策略：试用DeFi Saver的投资组合系统或Recipe Creator。

4、Ondo Finance

描述：Ondo Finance是一个支持“去中心化金融风险市场”的协议。

资源：网站‌|文档‌| Twitter‌ | Discord‌

理由：Ondo团队之前曾说过“Ondo目前没有代币”，注意一下，他们的说法是当时没有。

空投策略：加入Ondo的一个风险市场（请注意，他们有特定的“开放窗口”，在此期间你才可以加入）。

5、Saddle Finance

描述：Saddle是一个去中心化交易所，其对稳定币和封装BTC等锚定加密资产进行了优化。

资源：网站‌|文档‌| Twitter‌ | Discord‌

理由：Saddle是Curve系统的一个改编版本，尽管它是用Solidity语言而不是Vyper编写的，考虑到Curve推出了CRV，不难想象Saddle最终也会有自己的代币。

空投策略：通过该交易所进行代币互换，或者充当流动性提供者（LP）。

6、Charm

描述：Charm协议提供了一套 DeFi 产品，包括 Alpha Vaults、Cube Token等。

资源：网站 ‌| 文档‌ |Twitter|Discord

理由：从第一天起，Charm 团队的既定目标就是“任何人都可以研究、构建和治理 Charm 中的协议”。这里提到了“治理协议”，似乎意味着他们将发行一种代币。

空投策略：存入 Charm 的初始产品之一，例如Alpha Vaults。

7、Slingshot

描述：Slingshot 是目前在 Polygon 上运行的一个 Web3 交易平台，接下来它还将支持 Arbitrum。

资源：网站‌ | Twitter|Discord

理由：Slingshot 似乎可能会走其他 DEX走过的道路，并最终发布自己的代币。

空投策略：连接到 Polygon 网络并在 Slingshot 应用上进行一些交易。

8、TokenSets

描述：TokenSets 是一个建立在 Set 协议之上的 DeFi 资产管理平台。

资源：Website|Docs|Twitter|Discord

理由：该项目似乎要实现完全去中心化治理，这可能需要发布一个原生代币。

空投策略：购买/出售Set token或创建你自己的Set token。

9、Gro Protocol

描述：Gro Protocol是一种迎合稳定币的收益聚合器协议。

资源：Website|Docs|Twitter|Discord

理由：Gro 团队计划让该项目“在不久的将来完全去中心化”，并计划让早期支持者参与治理。

空投策略：使用 Gro Protocol 的 Vault 或 PWRD 产品。

10、Cowswap

描述：Cowswap 是一个抗 MEV 的 DEX 聚合器，它构建在 v2 版本的Gnosis协议之上。

资源： Website | Docs | Twitter | Discord

理由：Cowswap 计划在完全去中心化的道路上建立一个 DAO，这意味着它会有一个代币。

空投策略：通过该交易所进行一些交易。

11、Cozy Finance

描述：Cozy Finance 是一种为 DeFi 创建信任最小化保护市场的协议（即保险协议）。

资源：Website|Docs|Twitter|Discord

理由：迄今为止，大多数 DeFi 保险项目都推出了自己的代币，因此我们最终可能会在Cozy 身上看到同样的情况。

空投策略：在 Cozy 上借用/提供资产，或者，更高级的用户可以创建自己的保护市场。

12、Zapper

描述：Zapper 是一个面向 DeFi 和 NFT 用户的 Web3 钱包管理仪表板。

资源：Website | Twitter | Discord

理由：Zapper 还没有“消除”发行代币的可能性，也就是说，我们还有机会。

空投策略：试用Zapper原生的Exchange、Pool、Save等产品。

13、Zerion

描述：Zerion 是另一个面向 DeFi 和 NFT 用户的Web3 钱包管理仪表板。

资源：Website|Twitter|Discord

理由：和Zapper 一样，我们可能会看到 Zerion 在未来某个时候推出一个原生代币来促进社区治理。

空投策略：试用Zerion的Invest、Exchange、Pool产品等。

14、 Kwenta & Lyra

描述：这是两个不同的协议，但它们的共同点是，两者都是构建在 Synthetix 上的衍生项目，并且已部署在 Optimism L2 上。

资源：Kwenta Twitter|Lyra Twitter

理由：这两个项目方已公开确认将进行追溯性空投。

空投策略：将 SNX 放在 Optimism 上，然后使用其中的任何一个协议。

15、 Mirror

描述：Mirror 是一个去中心化的内容发布平台

资源：Website|Twitter|Discord

理由：虽然 Mirror 已经拥有 WRITE代币，但该代币的发布是受到限制的，也许 Mirror 可能向其早期用户空投更多的代币。

空投策略：利用Mirror的各种工具，例如通过平台发布博客文章。

16、以太坊域名服务（ENS）

描述：ENS 是一个 Web3 域名和配置系统。

资源：Website|Docs|Twitter|Discord

理由：ENS正在演变成一个成熟的去中心化身份系统。未来，由代币持有者驱动的去中心化治理可能会监督这个系统。

空投策略：注册/更新ENS域名。

17、Optimism

描述：Optimism 是一种Optimistic Rollup二层网络，其可以帮助以太坊实现扩容。

资源：Website|Docs|Twitter|Discord

理由：似乎rollup二层网络都会推出自己的代币来引导流动性并争夺用户。而Optimism是一个需要关注的大项目。

空投策略：将资金从以太坊转移到 Optimism，并使用一些应用，例如Uniswap 或 Synthetix。

18、Arbitrum

描述：和Optimism一样，Arbitrum也是一个optimistic rollup二层项目，其最近的发展势头非常不错。

资源：Website|Docs|Twitter|Discord

理由：再强调一下，作为一个rollup 二层项目，Arbitrum 似乎最终可能会推出自己的代币来引导流动性并争夺用户。

空投策略：将资金从以太坊主网转移到Arbitrum，并使用Balancer等应用。

19、zkSync

描述：zkSync 是使用zk rollup技术的L2扩容网络。

资源：Website|Docs|Twitter|Discord

理由：zkSync 的创建者 Matter Labs 已确认他们将发布代币以激励早期用户。

空投策略：打开zkSync 钱包并测试这个L2网络。

20、PartyBid

描述：PartyBid 是一个用于“对 NFT进行集体竞标”的dapp，该项目让一群朋友甚至一群陌生人可以轻松地一起竞标 NFT。

资源：Website|Twitter|Discord

空投策略：针对你想要参与的任何NFT发起PartyBid，或向已激活的PartyBid 贡献 ETH（如果竞价失败，你可以取回自己的ETH）。

21、Foundation

描述：Foundation 是一个面向创造者经济的 NFT 铸造和市场平台。

资源：Website|Twitter

理由：在 SuperRare 通过 RARE 推出其社区所有的治理模式之后，像Foundation 这样的类似平台似乎很可能会选择类似的方法。

空投策略：在 Foundation 上竞标/收集 NFT，请记住，不要因为纯粹的投机原因而买入自己不喜欢的NFT。

22、Zora

描述：Zora 是一个 NFT 铸造和市场协议。

资源：Website|Docs|Twitter|Discord

理由：社区管理和治理是许多 NFT 平台的未来，因此如果Zora 最终发行自己的代币，我不会感到惊讶。

空投策略：在 Zora 上铸造和收集 NFT。

23、 Fractional

描述：Fractional 是一个 NFT 碎片化平台，例如，你可以将一个稀有的 1/1 NFT 分成 1,000 个碎片。

资源：Website|[Docs](https://docs.fractional.art/fractional/#:~:text=Fractional is a decentralized protocol,the NFT that they own.)|Twitter|Discord

理由：如果我没记错的话，Fractional 团队对未来发行原生代币的可能性表示出了开放的态度。

空投策略：在平台上收集 NFT 的碎片，或者自己拆分NFT。

24、 Sudoswap

描述：Sudoswap 是一个交易 NFT 和 ERC20代币的交易平台。

资源：Website|Twitter|Discord

理由：Sudoswap 团队富有创新精神和想象力，他们似乎可能会在某个时候进行一次深思熟虑、前所未有的空投。

空投策略：通过 Sudoswap 应用程序进行一些 NFT 交易。

25、OpenSea

描述：OpenSea 目前是全球领先的 NFT 市场。

资源：Website|Docs|Twitter|Discord

理由：随着大型新玩家（例如Coinbase 和 Reddit）持续进入 NFT 领域，OpenSea 可能会推出自己的代币以激励用户留在 OpenSea 上。

空投策略：通过 OpenSea 铸造和收集 NFT。

结论

请务必记住，使用年轻的 DeFi 和 NFT 项目可能存在着风险，它们本质上都是实验。

因此，在深入探索这些协议之前，你需要做更多的研究。 如果你确实测试了其中的任何一个，请确保谨慎进行，并且仅使用自己能够承受损失的资金。

也就是说，正如 Darren Lau 在之前的推文中提到的那样，即使是适度的参与，也可以让你获得丰厚的空投回报。

最后，请记住，我们最终可能会看到像 Arbitrum 和 Optimism 这样的rollup项目推出自己的原生代币。

另外，你可以尝试像 Hop Protocol 这样的项目，你可以在其中与这些rollup网络进行交互，这可能会让你实现“一石二鸟”。
