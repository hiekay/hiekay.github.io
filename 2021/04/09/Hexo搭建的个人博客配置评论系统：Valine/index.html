一、注册Leancloud

评论系统其实是放在Leancloud上的，因此首先需要去注册一个账号

Leancloud官网，点我注册

注册完以后需要创建一个应用，名字可以：blog ，然后 进入应用->设置->应用key

获取你的appid 和 appkey 如图所示：

二、博客主题配置文件配置

拿到你的appid和appkey之后，打开主题配置文件 搜索 valine，填入appid 和 appkey

三、配置域名

记得在Leancloud -> 设置 -> 安全中心 -> Web 安全域名 把你的域名加进去
