Front-matter 是文件最上方以 --- 分隔的区域，用于指定个别文件的变量，举例来说：

hexo new page tags

title: tags
date: 2017-12-02 21:01:24
type: “tags”

hexo new page categories

title: categories
date: 2017-12-02 21:01:24
type: “categories”
`
